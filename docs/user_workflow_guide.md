# User Workflow Guide - Westpharma Order System

## 👤 For Standard Users
1.  **Create Orders**: Navigate to the Dashboard or Order list and click "Submit Order".
2.  **Manage Orders**: You can view your own orders. If an order is still "Pending", you have the authority to edit the details or delete it entirely.
3.  **Track Status**: Monitor the status badge on your orders. It will transition to "Approved" or "Rejected" once a Manager or Admin has reviewed it.

## 💼 For Department Managers
1.  **Review Queue**: Your Dashboard will alert you to orders awaiting review.
2.  **Approve Orders**: You can approve orders with an amount **less than $10,000**.
3.  **Limitations**:
    *   You cannot approve orders you created.
    *   High-value orders (≥ $10,000) are reserved for Administrators and will have approval actions disabled for you.
4.  **Reject Orders**: You can reject any order (except your own) to send it back for review.

## 👑 For System Administrators
1.  **Full Oversight**: You can see every order in the system regardless of who created it.
2.  **Global Approval**: You have the authority to approve orders of **any value**, including high-value procurement requests.
3.  **User Management**:
    *   Access the "User Directory" to view all system accounts.
    *   Provision new users with specific roles.
    *   Revoke access by deleting user accounts.

## 📊 Status Reference
- **Pending**: Order is submitted and waiting for review.
- **Approved**: Order has been finalized and is ready for procurement.
- **Rejected**: Order has been denied or needs significant changes.
